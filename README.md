Project Title
------------
DSPEditor

Description
------------
Simple audio editor for adding audio effects on samples

Features
------------
* **Adding audio effects on samples**
- Tremolo
- Echo
- Delay
- Wah-wah
- Much more...
* **Cuting audio samples**
* **Playing audio samples**
